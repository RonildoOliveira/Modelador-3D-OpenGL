# Modelador 3D

## Elementos
- Torus
```
Torus(float innerRadius, float outterRadius,
	  int slices, int stacks,
	  float tx, float ty, float tz,

 	  float sx, float sy, float sz);

```
Teapot(float tx, float ty, float tz,
